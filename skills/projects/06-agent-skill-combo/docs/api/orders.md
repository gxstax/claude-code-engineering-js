**Description**: List all orders belonging to the currently authenticated user. Orders are filtered by `req.user.id`.

**Response**:

### GET `/:id`

🔒 Requires authentication (requireAuth)

**Description**: Get detailed information about a specific order. The order must belong to the authenticated user; otherwise a 403 is returned.

**Parameters**:

| Name | In | Type | Required | Description |
|------|----|------|----------|-------------|
| id | path | string | true | The unique identifier of the order |

**Response**:

| Status | Description |
|--------|-------------|
| 200 | The requested order object |
| 403 | Forbidden — order belongs to another user |
| 404 | Order not found |

**Response Example**:

```json
{
  "data": {
    "id": "101",
    "userId": "1",
    "status": "pending",
    "total": 29.99,
    "items": [
      { "productId": "p1", "quantity": 2, "price": 14.99 }
    ],
    "createdAt": "2025-06-20T10:00:00Z"
  }
}
```
